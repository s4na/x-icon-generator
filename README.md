## 概要

X Icon Generatorは、X（旧Twitter）のプロフィールアイコンを円形表示に最適化するためのWebツールです。画像をアップロードして、カスタマイズ可能な背景と組み合わせ、複数サイズでプレビューしながら最適なアイコンを作成できます。

## 主な機能

### 🎨 背景カスタマイズ
- **単色背景**: カラーピッカーで自由に色選択
- **グラデーション背景**: 線形・放射状グラデーション（角度調整可能）
- **パターン背景**: ストライプ、ドット、チェック、波模様
- **プリセット背景**: トレンドカラー配色（10種類）
- **透明度調整**: 背景とアイコンのブレンド具合を調整

### 📐 画像調整機能
- **拡大縮小**: スライダー、マウスホイール、ピンチイン・ピンチアウト対応
- **位置調整**: ドラッグで自由に移動（マウス・タッチ対応）
- **キーボード操作**: 矢印キーで微調整、+/-でズーム
- **グリッド表示**: 位置調整を補助する表示切り替え
- **リセット機能**: ワンクリックで初期状態に戻す

### 👁 マルチサイズプレビュー
- **プロフィールページ**: 200×200px（大サイズ）
- **タイムライン**: 48×48px（中サイズ）
- **リプライ**: 24×24px（小サイズ）
- **リアルタイム更新**: 調整が即座に全プレビューに反映

### 📥 ダウンロード
- **PNG形式**: 高品質、透明度対応
- **JPG形式**: ファイルサイズ小
- **出力サイズ**: 400×400px（Xの推奨サイズ）
- **背景込み**: アイコンと背景を合成した完成画像

### 📱 デバイス対応
- **レスポンシブデザイン**: PC、タブレット、スマートフォン完全対応
- **タッチ操作**: ドラッグ、ピンチズーム対応
- **マウス操作**: ホイールズーム、ドラッグ対応
- **キーボードショートカット**: 効率的な編集操作

## 使い方

### 1. 画像をアップロード
ファイル選択ボタンをクリックするか、ドラッグ&ドロップで画像をアップロードします。

**対応フォーマット**: JPG, PNG, GIF（最大10MB）

### 2. 画像を調整
- **拡大縮小**: スライダーで調整（50% - 200%）
- **位置調整**: ドラッグで画像を移動
- **マウスホイール**: PC環境でのズーム
- **ピンチ操作**: モバイル・タブレットでのズーム

### 3. 背景を選択
- **単色**: シンプルな単色背景
- **グラデーション**: 2色のグラデーション（線形・放射状）
- **パターン**: ストライプ、ドット、チェック、波模様
- **プリセット**: トレンドカラー配色から選択

### 4. プレビュー確認
3つのサイズでXでの実際の表示を確認：
- プロフィールページ（大）
- タイムライン（中）
- リプライ（小）

### 5. ダウンロード
PNG形式またはJPG形式で400×400pxの正方形画像をダウンロード。

## キーボードショートカット

| キー | 機能 |
|------|------|
| `↑` `↓` `←` `→` | 画像を移動（1px） |
| `Shift` + 矢印 | 画像を移動（10px） |
| `+` / `=` | 拡大 |
| `-` / `_` | 縮小 |
| `R` | リセット（初期状態に戻す） |
| `G` | グリッド表示切り替え |

## 技術スタック

- **HTML5**: セマンティックマークアップ
- **CSS3**: モダンレイアウト（Flexbox, Grid）、レスポンシブデザイン
- **Vanilla JavaScript (ES6+)**: フレームワーク不要
- **Canvas API**: 画像処理、プレビュー生成
- **File API**: 画像ファイル読み込み
- **Drag and Drop API**: ドラッグ&ドロップ対応

## ローカルでの実行

```bash
# リポジトリをクローン
git clone <repository-url>

# ディレクトリに移動
cd x-icon-generator

# シンプルなHTTPサーバーで起動（例：Python）
python -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```

または、`index.html` を直接ブラウザで開くことも可能です。
